## (i)

Let $`f(x) = c`$. Any open set $`S \in \tau_1`$ either contains $`c`$ or not.

If $`c \in S`$, then $`f^{-1}(S) = X`$.  
If $`c \notin S`$, then $`f^{-1}(S) = \emptyset`$.  

Hence continuous.

## (ii)

For any open set $`S \in \mathcal{T}`$, $`f^{-1}(S) = f(S) = S \in \mathcal{T}`$.

Hence continuous.
